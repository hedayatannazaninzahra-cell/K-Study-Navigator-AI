# 🇰🇷 K-Study Navigator AI

A smart AI-inspired web application designed to help international students plan their academic journey to South Korea.

## ✨ Features

- 🎓 University pathway exploration
- 📊 Student profile evaluation
- 🇰🇷 Korean language readiness analysis
- 🏆 Scholarship preparation guidance
- 🤖 AI-inspired admission score calculator

## 🛠 Technologies

- Python
- Flask
- HTML5
- CSS3

## 🎯 Purpose

This project was created to support students interested in studying in South Korea by providing a simple academic planning assistant.
